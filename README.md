# 端午 H5 小游戏

这是一个可直接打开运行的端午节接物小游戏原型。

## 玩法

- 游戏时长：60 秒
- 左右移动粽叶小船接住食材
- 红豆：+10 分
- 红枣：+10 分
- 糯米：+10 分
- 香蕉皮：-10 分

## 操作

- 手机：手指在游戏区域左右拖动
- 电脑：键盘 `←` `→` 或 `A` `D`

## 素材替换

把你的正式素材放到 `assets` 文件夹，并使用下面这些文件名：

- `assets/bg.jpg`：游戏背景图
- `assets/start-button.png`：开始按钮图
- `assets/boat.png`：粽叶小船
- `assets/rules.png`：底部积分规则整图
- `assets/red-bean.png`：红豆
- `assets/jujube.png`：红枣
- `assets/rice.png`：糯米
- `assets/banana-peel.png`：香蕉皮

如果某个素材暂时没放，页面会自动使用当前的简化样式或表情占位，不会影响试玩。

## 运行

直接双击 `index.html` 即可在浏览器中打开。
